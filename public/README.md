# AImum - 一人公司 AI 操作系统

🦁 **一个人活成一个团队**

## 🎯 产品定位

AImum = AI + 团队协作系统

为一人公司和创业者提供 AI 驱动的"虚拟团队"，通过搭积木的方式组装自己的 AI 部门。

## 🏗️ 核心功能

### 🤖 AI 对话
- 多模型切换（DeepSeek / MiniMax / Claude）
- 部门专业化（销售/市场/财务/行政/技术）
- 上下文记忆

### 🧩 插件市场
- 第三方插件支持
- 轻松扩展能力
- 开发者生态

### 💰 令狐充系统
- 令牌充值
- 按量计费
- 灵活定价

## 🚀 快速开始

### 1. 访问

打开浏览器，访问：**https://aimum.vercel.app**

### 2. 登录

使用手机号登录，立即获得 **20 个体验令牌**。

### 3. 选择部门

- 💼 销售部
- 📢 市场部  
- 💰 财务部
- 📋 行政部
- 💻 技术部

### 4. 开始对话

告诉 AI 你的需求，它会帮你完成工作。

## 📱 PWA 安装

手机访问后，点击"添加到主屏幕"，即可像原生 App 一样使用。
