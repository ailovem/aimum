## 快速开始

### 访问 AImum

打开浏览器访问: **https://aimum.vercel.app**

或直接打开本地文件:
```
D:\openwork\00_active\aimum\public\index.html
```

### 登录账号

1. 点击右上角 **"登录 / 注册"**
2. 输入手机号
3. 获取验证码（开发环境会显示验证码）
4. 输入验证码完成登录

### 开始使用

登录后，你可以：

- 💬 **AI 对话** - 与 AI 助手对话
- 💎 **Token 管理** - 查看余额和充值
- 🧩 **插件市场** - 安装插件扩展功能

## 功能介绍

### 💬 AI 对话

AImum 支持多种 AI 模型:

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| Claude Sonnet 4 | 最强能力 | 复杂任务、写作 |
| Claude Haiku 3 | 快速响应 | 简单问答 |
| GPT-4o | 通用能力强 | 各类任务 |
| GPT-4o Mini | 性价比高 | 日常对话 |
| DeepSeek | 便宜快速 | 代码任务 |

**使用技巧**:
- 点击侧边栏可切换模型
- 不同任务选择最适合的模型
- 代码任务推荐使用 DeepSeek

### 💎 Token 系统

Token 是 AImum 的使用货币。

#### Token 消耗

| 操作 | 消耗 |
|------|------|
| 普通对话 | 约 10-50 tokens/条 |
| 复杂分析 | 约 100-500 tokens/次 |
| 插件使用 | 5-20 tokens/次 |

#### 套餐

| 套餐 | 价格 | 包含 | 适合 |
|------|------|------|------|
| 免费版 | ¥0 | 100 tokens | 试用体验 |
| 专业版 | ¥29/月 | 5000 tokens | 日常使用 |
| 企业版 | ¥99/月 | 20000 tokens | 重度使用 |

#### 充值方式

- **Wise 国际汇款**
- **支付宝** (待开放)
- **微信支付** (待开放)

## 使用教程

### 教程 1: 基础对话

```
1. 登录账号
2. 点击 "新建对话"
3. 输入问题
4. AI 回复
```

### 教程 2: 使用代码专家

```
1. 在侧边栏选择 "代码专家" 角色
2. 或在模型选择器选择 "DeepSeek"
3. 输入代码相关问题
4. 获取代码建议和解释
```

### 教程 3: 文件转换

```
1. 打开插件市场
2. 安装 "文件转换专家"
3. 在对话中使用插件
4. 上传文件进行转换
```

### 教程 4: 充值 Token

```
1. 点击 "Token 管理"
2. 选择套餐（免费/专业/企业）
3. 选择支付方式
4. 完成支付
```

## 常见问题

### Q: Token 不够用怎么办？

**A**: 
- 免费试用：注册送 100 tokens
- 充值：选择合适套餐
- 节省：使用性价比高的模型（DeepSeek）

### Q: 验证码收不到？

**A**: 
- 开发环境：验证码会直接显示在页面上
- 检查手机号格式是否正确
- 确认网络连接正常

### Q: AI 回复很慢？

**A**: 
- 切换到更快的模型（Claude Haiku 3 或 DeepSeek）
- 检查网络连接
- 复杂问题会消耗更多时间

### Q: 插件无法使用？

**A**: 
- 确认插件已安装
- 检查插件是否已启用
- 确认 Token 余额充足

### Q: 如何删除账号？

**A**: 
- 目前暂不支持自助删除
- 联系客服处理
